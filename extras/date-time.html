Date and Time in WP

In WordPress you can use the functions the_date() and get_the_date() to print or retrieve the date on which the post was written.
Both functions can be used passing a string parameter $d to specify the PHP date format you like.

https://developer.wordpress.org/reference/functions/get_the_date/

Day of the month
d Numeric with leading 0 01-31
j Numeric without leading 0 1-31
s English suffixes st, nd or th in the 1st, 2nd o 15th
Day of the week
l Complete name Sunday - Saturday
D Short name Sun - Sat
Month
m Numeric with leading 0 01-12
n Numeric without leading 0 1-12
F Complete name January - December
M Short name Jan - Dec
Year
Y Numeric 4 digits 1999,2003
y Numeric 2 digits 99,03
Hours
a Lowercase am,pm
A Uppercase AM,PM
g 12 hours without leading 0 1-12
h 12 hours with leading 0 01-12
G 24 hours without leading 0 0-23
H 24 hours with leading 0 00-23
i Minutes with leading 0 00-59
s Seconds with leading 0 00-59

Examples

echo get_the_date( "d/m" ) prints 08/12
echo get_the_date( "j/m/y - H:i" ) prints 8/12/19 - 22:48
echo get_the_date( "D d F Y" ) prints Tue 16 December 2019
